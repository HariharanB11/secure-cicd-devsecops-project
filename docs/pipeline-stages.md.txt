# 🚀 Pipeline Stages Explained

## GitLeaks
Detects secrets like API keys and passwords.

## SonarQube
Analyzes code quality and detects bugs.

## OWASP Dependency Check
Finds vulnerabilities in third-party libraries.

## Trivy Image Scan
Scans Docker image for vulnerabilities.

## Trivy FS Scan
Scans project filesystem for issues.

## Deployment
Deploys application using Docker Compose.
